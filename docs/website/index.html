PETSc 3.0 Alpha

We are currently developing the next generation of PETSc software. This website provides access to the software and (limited) documentation; the current supported version of PETSc is located at http://www.mcs.anl.gov/petsc. You should only use PETSc 3.0 if you are very adventurous and willing to experiment with incomplete, poorly documented (but very exciting) software.

PETSc 3.0 is implemented using the Scientific Interface Definition Language (SIDL) and compiler developed by the LLNL Babel team

 

The goals of the PETSc 3.0 include

  • support for many languages (C++,Python,F90,...) and environments (Matlab, Mathematica, ...)
  • support for mixing numerical precision: single, double, complex and double complex

PETSc 3.0 requires that you first install

  • for Microsoft Windows, the Cygwin tools
  • Python 2.1 or later (it is likely Python is already installed on your system)
  • the BitKeeper software control system
  • MPI, if it is not installed on your system we suggest MPICH

To install PETSc 3.0 save this file bootstrap.sh and run it in a (if Windows Cygwin bash) shell with the command python bootstrap.sh

If you have any problems please send mail to petsc-maint@mcs.anl.gov and include all output
